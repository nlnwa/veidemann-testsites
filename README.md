# Testsites for Veidemann

## Dependencies
- skaffold
- minikube

## Setup
 - [Install skaffold](https://skaffold.dev/docs/install/)
 - Clone this repo

## Run veidemann-testsites in your minikube-cluster
```
 skaffold dev 
```
## View testsites in a browser
```
 minikube service veidemann-testsites
```
